# 📖 Tentang Project

AquaGas merupakan aplikasi penjualan Air Galon Isi Ulang, Air Mineral (AQUA), dan Gas LPG 3KG berbasis website. Sistem ini dibuat untuk mempermudah proses pembelian produk secara online, mulai dari pemesanan, pembayaran, hingga monitoring pesanan.

Website ini memiliki tiga hak akses utama yaitu **Buyer**, **Seller**, dan **Administrator** sehingga seluruh proses bisnis dapat dikelola secara terstruktur.

# 🚀 Cara Menjalankan Project

## 1. Clone Repository

```bash
git clone https://github.com/USERNAME/AquaGas.git
```

## 2. Pindahkan Project

Simpan project ke folder

```
htdocs
```

contoh

```
C:\xampp\htdocs\TugasBESARPemweb
```

---

## 3. Jalankan XAMPP

Aktifkan

- Apache
- MySQL

---

## 4. Import Database

Buka phpMyAdmin

Buat database

```
aquagas
```

Lalu import file

```
database.sql
```

---

## 5. Jalankan Website

```
http://localhost/TugasBESARPemweb/
```
